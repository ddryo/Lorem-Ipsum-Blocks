A plugin that makes it easy to use blocks with dummy text.

== Description ==

This plugin adds variations to some core blocks and makes it easy to use blocks with dummy text and dummy images already entered.

It's very lightweight because it just adds a "variation" of the core block.

== Available Blocks ==

- Lorem Paragraph
- Lorem Paragraph (Long)
- Lorem Heading
- Lorem List
- Lorem image
- Lorem Columns
- Lorem Media & Text


== Installation ==

This plugin can be installed directly from your site.

1. Log in and navigate to "Plugins" â†’ "Add New".
2. Type "Lorem Ipsum Blocks" in the search field and press Enter.
3. Locate the plugin in the list of search results and click "Install Now".
4. Once installed, click the "Activate" button.
